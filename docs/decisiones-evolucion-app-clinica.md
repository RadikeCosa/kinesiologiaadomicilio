# Decisiones de dirección — evolución a app clínica conviviente

> Estado: vigente
> Fecha: 2026-04-16 (UTC)

## Decisiones registradas

1. **La landing pública se mantiene** como superficie activa de captación local.
2. Se acepta la **evolución futura** hacia una app clínica que conviva en el mismo repositorio.
3. Esa evolución se entiende como dirección de producto; **no implica implementación actual**.
4. El foco funcional inicial previsto para la futura app clínica es:
   - paciente (alta y base mínima),
   - contacto principal / quién escribe,
   - EpisodeOfCare / tratamiento activo,
   - visitas / encuentros.
5. Quedan explícitamente fuera del alcance inicial previsto:
   - agenda,
   - pagos,
   - self-booking / autogestión de turnos,
   - multiusuario y auth compleja,
   - panel administrativo completo.

## Alcance del documento

Este documento registra decisiones de encuadre ya tomadas.
No es roadmap, backlog ni plan técnico de implementación.
