# Archivo documental (`docs/archive`)

> Estado: vigente
> Última actualización: 2026-06-01 (UTC)

Este directorio conserva documentación histórica o cerrada para mantener trazabilidad sin mezclarla con la documentación activa del proyecto.

## Subcarpetas

- `auditorias/`: auditorías puntuales cerradas o superadas.
- `planes-cerrados/`: planes de implementación que ya no son guía activa.
- `cierres/`: documentos de cierre históricos de iniciativas específicas.
- `historico-fhir/`: planes, backlog, auditorías, decisiones transicionales y hardenings FHIR ya cerrados o superados por la fuente operativa vigente.
- `normalization-data/`: planes y cierres de normalización administrativa ya implementados.
- `producto/auditorias/`: auditorías de producto/UX ya cerradas o absorbidas por documentación vigente.
- `producto/planes-cerrados/`: planes de producto implementados o reemplazados por la fuente operativa.

## Regla de uso

Si un documento archivado vuelve a ser relevante, debe referenciarse explícitamente desde un documento activo (por ejemplo `docs/fuente-de-verdad-operativa.md` o `docs/fhir/README.md`) indicando el contexto.
